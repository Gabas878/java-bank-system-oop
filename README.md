# 💻 Sistema Bancário em Java

Projeto desenvolvido com foco em prática de lógica de programação, Programação Orientada a Objetos (POO) e organização em camadas.

---

## 🚀 Funcionalidades

- Cadastro de usuários (nome, email, CPF e senha)
- Login com validação de credenciais
- Criação de conta bancária vinculada ao usuário
- Depósito em conta
- Saque com validação de saldo
- Transferência entre contas

---

## 🧠 Tecnologias e conceitos aplicados

- Java
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Separação de responsabilidades
- Manipulação de listas
- Validação de dados
